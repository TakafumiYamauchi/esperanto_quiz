## A. 最重要修正: サーバ応答メッセージの日本語リーク解消（11メッセージ×3言語）

**問題**: app.js が `result.message ||`（サーバ文字列を訳語より優先）するため、
中韓ユーザーのスコア保存・ランキング表示で**日本語がそのまま表示**されていた
（保存成功「ランキングに保存しました。今回の38.5点を…」等、計11メッセージ）。

**修正**: PWAの payload は既に `targetLang` を送信していたため、**サーバ側のみ**で解決:
- `mobile_score_sync.py`: 6メッセージの ja/zh/ko 表＋`_msg()` ヘルパー（points書式対応）
- `mobile_ranking.py`: 5メッセージ（劣化モード警告2種含む）の同様の表
- 設計: モジュール内完結・新規クロスモジュールimportなし（stale-module-cache由来の
  ImportError リスクを構造的に回避）。旧クライアント・不正値は ja フォールバック。
- 検証: 3言語×全分岐のオフラインテスト＋フォールバック（fr/None/非dict→ja）合格。

## B. 中国語UI 約32箇所（日本語語彙のリーク・機械翻訳調・意味エラー）
- **意味エラー**: 「得分系数比单词版高约2.0倍」→「约为单词版的2.0倍」×2 —
  中国語の「比…高2倍」は**3倍**の意味になる得点説明の実害級誤訳
- **日本語リーク**: 殿堂→名人堂 / 件→条（助数詞）/ 本篇→正式部分 ×2 / 画面→页面 /
  保存容量→存储空间 ×4 / 不明→未知 / 保存中的测验→已保存的测验 / 時点→保存时
- **機械翻訳調・文法**: 已只清除→仅清除了…没有被删除 / 要执行吗→确定要清除吗 /
  无论出题方向→…如何 / 正在暂时显示→暂时显示 ×5 / 全部题目后→做完全部题目后 /
  按通常的→按正常得分的 ×2 / 只有…失败→但…失败
- **用語統一**: 取得→获取 ×7 / 问题→题目 / 对应词→相关词（相関詞の中国語Esperanto界標準）×2
- **誤訳**: rankingUnavailable が保存说明の複製だった→排行榜显示…に修正
